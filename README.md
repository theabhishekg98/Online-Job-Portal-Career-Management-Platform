##  Steps to run the application
- Prerequisite: node js, kafka
- Download the reposoitory
- Follow the commands below to install node packages
 ```sh
cd client
npm install

cd server
npm install

cd kafka-server
npm install
```

- Follow the commands below to start the application
 ```sh
cd client
npm start

cd server
nodemon

cd kafka-server
nodemon
```
